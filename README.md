# Obsyk Documentation

User documentation for [Obsyk](https://obsyk.ai) - Observability & Security for AI Workloads.

## Local Development

### Prerequisites

- Python 3.9+
- pip

### Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally with live reload
mkdocs serve

# Build static site
mkdocs build
```

The local server runs at http://localhost:8000
